### Listas
#### Listas Desordenadas
Pueden usar guines, simbolo de suma o asteriscos.
- Elemento 1
+ Elemento 2
* Elemento 3

#### Listas Ordenadas
Solo se pueden usar numeros en estas listas.
1. Elemento 1
2. Elemento 2
3. Elemento 3

#####  Listas Anidadas
1. Elemento 1
    - Elemento 1
    + Elemento 2
2. Elemento 2
    * Elemento 1
